# 🎓 Career Navigator — Платформа для профориентации и трудоустройства

Привет! Это мой дипломный проект — **Career Navigator**.
Это веб-платформа, которая решает важную проблему: помогает абитуриентам и студентам выбрать правильный образовательный путь, а затем найти работу мечты.

Система объединяет три ключевые стороны:
- **Абитуриенты/Студенты** — проходят профориентацию, ищут вузы и вакансии
- **Вузы (HEI)** — публикуют информацию о программах, требованиях и условиях поступления
- **Работодатели** — размещают вакансии, стажировки и ищут молодых специалистов

---

## 🏗 Архитектура проекта

Проект построен по классической **клиент-серверной архитектуре** с разделением на Backend и Frontend:

### Backend (Node.js + Express)
Серверная часть реализована на **Express.js** с использованием **MVC паттерна**:
- **Models** — работа с данными (пользователи, студенты, вузы, работодатели, профориентация)
- **Controllers** — бизнес-логика и обработка запросов
- **Routes** — маршрутизация API endpoints
- **Views** — серверные шаблоны на **EJS** (для рендеринга страниц на сервере)

### Frontend (Nuxt.js + Vue 3)
Клиентская часть построена на современном стеке:
- **Nuxt.js 4** — SSR/SSG фреймворк для Vue.js
- **Vue 3** — реактивный UI
- **TypeScript** — типизация для надежности кода
- **Axios** — HTTP-клиент для общения с Backend API

## 🚀 Как запустить проект

### 1. Клонирование репозитория
```bash
git clone https://github.com/MrNikto1/Career-Navigator.git
cd Career-Navigator
```

### 2. Настройка Backend

Перейди в папку Backend и установи зависимости:
```bash
cd Backend
npm install
```

Создай файл `.env` и настрой переменные окружения:
```env
PORT=3000
DB_HOST=localhost
DB_USER=your_username
DB_PASSWORD=your_password
DB_NAME=career_navigator
JWT_SECRET=your_secret_key
```

Запусти сервер:
```bash
npm start
# или для разработки с автоперезагрузкой:
npm run dev
```

Backend будет доступен на `http://localhost:3000`

### 3. Настройка Frontend

Открой новую терминал и перейди в папку frontend:
```bash
cd frontend
npm install
```

Запусти dev-сервер Nuxt:
```bash
npm run dev
```

Frontend будет доступен на `http://localhost:3000` (или другом порту, если 3000 занят)

### 4. Настройка базы данных

Не забудь:
1. Создать базу данных `career_navigator`
2. Применить миграции (если есть) или создать таблицы вручную
3. Заполнить начальные данные (опционально)

## 📡 API Endpoints

Backend предоставляет RESTful API для взаимодействия с Frontend:

### Пользователи (`/api/users`)
- `POST /register` — регистрация нового пользователя
- `POST /login` — авторизация
- `GET /profile` — получение профиля
- `PUT /profile` — обновление профиля

### Студенты (`/api/students`)
- `GET /` — список студентов
- `GET /:id` — информация о студенте
- `POST /` — создание профиля студента

### Вузы (`/api/HEI`)
- `GET /` — список вузов
- `GET /:id` — информация о вузе
- `POST /` — добавление вуза (для админов)
- `GET /programs` — список образовательных программ

### Работодатели (`/api/employers`)
- `GET /` — список работодателей
- `GET /:id` — информация о компании
- `GET /vacancies` — список вакансий
- `POST /vacancies` — создание вакансии

### Профориентация (`/api/career-guidance`)
- `GET /tests` — список тестов
- `POST /test/submit` — отправка результатов теста
- `GET /recommendations` — получение рекомендаций
